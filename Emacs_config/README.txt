Рекомендации по настройки emacs.
Для Windows:
1. Скопировать файл .emacs в папку C:\Users\username\AppData\Roaming
2. Сюда же скопировать папку .emacs.d с автодополнением и другими вещами как в
   IDE.
Для Unix:
1. .emacs в домашнюю директорию: /home/%username%/
2. папку .emacs.d в домашнюю директорию: /home/%username%/
